# Dynamic Timetable 📅

A sleek, responsive, and interactive timetable application that automatically highlights the current period based on the time of day.

View The Project - (https://gokul-codes-420.github.io/Dynamic-Timetable-Schedule/)

## ✨ Features

- **Real-time Highlighting**: Automatically detects and highlights the current ongoing class.
- **Interactive Tabs**: Easy navigation between different days of the week.
- **Modern UI**: Clean, minimalist design with a professional color palette.
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop screens.
- **Detailed Information**: Shows subject, faculty name, and time slot for each period.

## 🚀 Getting Started

To run this project locally, simply clone the repository and open `index.html` in your web browser.

```bash
git clone https://github.com/gokul-codes-420/Dynamic-Timetable-Schedule.git
cd Dynamic-Timetable-Schedule
open index.html
```

## 🛠️ Built With

- **HTML5**: For semantic structure.
- **CSS3**: For modern styling and responsiveness.
- **JavaScript**: For dynamic rendering and real-time period tracking.
